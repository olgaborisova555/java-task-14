INSTRUCTION Предоставляет информацию об объеме кода, который был выполнен или пропущен. Эта метрика полностью независима от источника 
форматирования и всегда доступна, даже при отсутствии отладочной информации в файле класса.
LINE JaCoCo Вычисляет покрытие линий для классов и исходного файла на основе фактических исходных покрытий линиями.
BRANCH Эта метрика подсчитывает общее число ветвей в методе и определяет количество выполненных или пропущенных ветвей.
Покрывает ветки.
COMPLEXITY Служит показателем для количества единичных тестовых случаев чтобы полностью покрыть определенную 
часть программного обеспечения.
